# json-server-ezwallet

## Install dependencies
` npm install `

## Run 
`json-server --watch db.json`

## Routes

HTTP requests

```
GET    /courses
GET    /courses/1
POST   /courses
PUT    /courses/1
PATCH  /courses/1
DELETE /courses/1
```
